## 文件说明

1. core


事件循环核心代码，负责把事件分发给对应的连接处理器。

2. net

连接处理器代码，负责读取字节流到接收缓冲区，不做协议解析。

3. protocol

应用层协议解析代码，定义了应用层数据格式，负责将字节流解析为应用层数据结构。

4. service

业务逻辑处理代码，负责按照业务需求处理应用层数据结构并转发给broker。

5. infra

中间件代码。

6. third_party

第三方库代码。
